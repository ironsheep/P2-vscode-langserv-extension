# REF-LangUpdates

Reference documents for Spin2/PASM2 language additions by PNut compiler version.

Each file covers the new language features introduced in a specific PNut release, documenting syntax, parameters, and examples. These are used to keep the extension's semantic parser, hover documentation, and highlighting current with the evolving language specification.

## Files

|------|-------------|---------------|

| V47 | v47 | Preprocessor directives, multitasking methods, PASM2 enhancements |

| V49 | v49 | Structure export/import across objects, compiler limit increases |
